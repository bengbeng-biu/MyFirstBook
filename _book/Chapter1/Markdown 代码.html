Markdown 代码

如果是段落上的一个函数或片段的代码可以用反引号把它包起来（`），例如：

`printf()` 函数

显示结果如下：

printf() 函数

代码区块

你也可以用 ``` 包裹一段代码，并指定一种语言（也可以不指定）：

​```javascript
public class HelloWorld{
    public static void main(String[] args){
        System.out.println("HelloWorld!");
    }
}
​
